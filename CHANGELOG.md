# Changelog

All notable changes to Speech Translate Rev will be documented in this file.

This project follows a pragmatic changelog format inspired by Keep a Changelog. Version numbers for Python package metadata use standard Python versions; GitHub release tags may include a `-rev` suffix.

## [0.1.0-rev] - 2026-06-23

### Added

- Established Speech Translate Rev as an independent, heavily modified derivative of Dadangdut33/Speech-Translate.
- Added modern WebView UI, shell, realtime, file, model, and settings workflows.
- Added detached transcription and translation window controls.
- Added interface language support for Simplified Chinese and English.
- Added runtime/controller tests and WebView contract coverage.
- Added GitHub community files, CI workflow, and project attribution.

### Changed

- Updated public branding, package metadata, build names, installer metadata, and repository links to Speech Translate Rev.
- Kept the Python import package name `speech_translate` for compatibility.
