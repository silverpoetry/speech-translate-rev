# Support

For bugs, feature requests, and usage questions, use GitHub Issues:

- Bug reports: https://github.com/silverpoetry/speech-translate-rev/issues/new?template=bug_report.md
- Feature requests: https://github.com/silverpoetry/speech-translate-rev/issues/new?template=feature_request.md

Before opening an issue:

- Search existing issues.
- Confirm you are using the latest release or current `main`.
- Include logs, screenshots, operating system, Python version, install method, and clear reproduction steps when reporting bugs.

Security-sensitive reports should follow `SECURITY.md`.
